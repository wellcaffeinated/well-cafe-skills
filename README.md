# well-cafe-skills

Personal Claude Code skills for wellcaffeinated.

## Install

```bash
/plugin marketplace add wellcaffeinated/well-cafe-skills
/plugin install obsidian-workflow@well-cafe-skills
```

## Skills

### obsidian-workflow

Working with the Main Vault — note creation, bookmarks, projects, areas, and vault organisation. Reads `CLAUDE.md` at the vault root for conventions before any vault work.
